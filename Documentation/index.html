A. Files Structure - top

Every file (HTML, CSS, JS) that was created by us has clear code, wrapped with comments - all this for your convenience.

  • css - CSS styles folder
  • fonts - webfont icons folder
  • img - folder for images
  • js - JavaScript files folder
  • php - PHP contact script folder
  • index.html - main HTML file

1. HTML Files and Structure

This Template has a Responsive layout and is based on the Bootstrap Framework. For more information about this visit Bootstrap site.

The default Bootstrap grid system utilizes 12 columns, making for a 940px wide container without responsive features enabled. With the responsive CSS file added, the grid adapts to be 724px and 1170px wide depending on your viewport. Below 767px viewports, the columns become fluid and stack vertically.

index.html is the main template file that contains the HTML structure of the template. Here is the general structure (shortened):

					
<!DOCTYPE html>
<html>
<head>
	<!-- Included CSS Files -->
</head>
<body>
         <!-- Preloader -->

            <!-- About Section -->

</body>
</html>

C. Customization - top

You can simply change template's, make the following changes.

1. Replace Navbar Title

To change the Navbar Title replace the Flatfy text located in the template's index.html file with yours (line 60).


<a href="#page-top" class="navbar-brand page-scroll">Flatfy</a>

2. Edit Text

All static texts can be modified within the index.html file. For example, to change the text "Hi, I’m John Doe" open index.html and edit text on line 86.

3. Change Status Numbers

To change the Status Numbers of your site open custom.js file within the js folder and edit the numbers on lines 73 to 91:


    $('.total-number-1').animateNumber({
            number: 1650, //change value here
            numberStep: comma_separator_number_step
    }, 2000);
        
    $('.total-number-2').animateNumber({
            number: 216, //change value here
            numberStep: comma_separator_number_step
    }, 2000);
        
    $('.total-number-3').animateNumber({
            number: 142, //change value here
            numberStep: comma_separator_number_step
    }, 2000);
        
    $('.total-number-4').animateNumber({
            number: 28, //change value here
            numberStep: comma_separator_number_step
    }, 2000);

5. Change InView Animations

You can define a simple animation for any element which will be fired when element appears on screen. List of all animations can be found here.

To define an InView animation, add a wow inview animated attribute with desired animation's name to any element. For example:


<p class="wow fadeInUp animated">Lorem ipsum dolor sit amet, consectetuer adipiscing elit</p>

6. Change the Email Address for Contact Form

Make sure that you change the default email address yourname@yourdomain.com to your own in contact_me.php file (line 19) within the php folder to receive emails when the contact form is submitted.


// Create the email and send the message
$to = 'yourname@yourdomain.com'; // Add your email address inbetween the '' replacing yourname@yourdomain.com - This is where the form will send a message to.
$email_subject = "Website Contact Form:  $name";
$email_body = "You have received a new message from your website contact form.\n\n"."Here are the details:\n\nName: $name\n\nEmail: $email_address\n\nPhone: $phone\n\nMessage:\n$message";
$headers = "From: noreply@yourdomain.com\n"; // This is the email address the generated message will be from. We recommend using something like noreply@yourdomain.com.
$headers .= "Reply-To: $email_address";	
mail($to,$email_subject,$email_body,$headers);
return true;			
?>
